START APPLICATION
--------------

For start the application start a local PHP sever and a MySQL database.
On database execute the SQL script /api/db/NAME_DB_db_schema.sql
Put your project in root folder of your PHP server.

Go to http://localhost

--------------
CONFIGURE
--------------

For PHP and database config edit
    /api/properties.php

For AngularJS edit
    properties.js
